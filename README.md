# Sendai CTF 2017 Web problem

## How to use
### Install docker compose

[Please read this page](https://docs.docker.com/compose/install/#install-compose)

### Start process

```
docker-compose build
docker-compose up -d
```

### Stop process

```
docker-compose stop
```
